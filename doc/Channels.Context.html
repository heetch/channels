    channels v0.0.1 Channels.Context behaviour

    This module specifies the API a Context is required to implement.

    A context is a module with a setup/2 function that receives and adapter and a configuration that specifies what setup an AMQP related process needs in order to work (declare exchanges, bind queues, etc),

    See Channels.Context.Spec for the details about de format of the specification.
